# hfss-patch-antenna
2.4 GHz Patch Antenna Design using Ansys HFSS
This repository presents the design and simulation of a **1×2 microstrip patch antenna array** operating at **2.4 GHz**, developed using **Ansys HFSS**.

## Antenna Specifications
- Operating Frequency: 2.4 GHz  
- Antenna Type: 1×2 Microstrip Patch Antenna Array  
- Substrate Material: FR4  
- Dielectric Constant (εr): 4.4  
- Substrate Thickness: 1.6 mm  
- Feeding Technique: Microstrip Line Feeding
